# gr_osview
Graphical operating system view

A clone of the SGI gr_osview utility.

## Installation

### Linux
Install Python 3 and TCL/TK toolkit:
```
sudo apt-get update
sudo apt-get install python3.8 python3-tk
```

Get this code:
```
git clone git@github.com:fdachille/gr_osview.git
```

Run:
```
cd gr_osview
python3.8 main.py
```
